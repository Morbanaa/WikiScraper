# WikiScraper
Pull From Wiki Search Results and Store and retrieve from SQL Database 

## 📌 Project Overview

Wiki Scraper is a Python-based terminal application that allows users to:

- Search for Wikipedia articles
- Store selected articles in a MySQL database
- View saved article summaries
- Delete stored articles
- Manage a persistent local article collection

This project demonstrates integration between:

- Python
- Wikipedia API
- MySQL
- SQL database operations
- Exception handling
- Cross-platform terminal support

---

## 🛠 Technologies Used

- wikipedia (Python wrapper for Wikipedia API)
- mysql-connector-python
- MySQL Server
- BeautifulSoup4 (dependency support)
